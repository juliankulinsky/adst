Recommendations for Cinema’s-Best-Event¶

Data source¶

Source: IMDB; The data was collected by IMDB and is two years old at the current time, so was collected in 2020.

Data characterisitcs¶

The dataset contains information about movies. In total there are 1000 observations and 16 columns. For each movie the data includes the following features: Link to the movie poster, title, release year, certificate, runtime, genre, IMDB rating, overview, meta score, director, stars, votes and gross earning.

Question¶

Which movies were most popular in the past years? Which genre was most popular? -> Select the best movies for a Cinemas-Best-Event

Data source¶

Source: IBM (Kaggle)

Data characterisitcs¶

The dataset contains information about employee attrition and performance. In total there are 1470 observations and 35 columns. For each employee the data includes information about the age, department, education, gender, hourly rate, job level and job role etc. Further features can be retrieved by executing employees.info()

Question¶
